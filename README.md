# Prompts Hub

A centralized collection of AI prompts for various use cases, organized by category and topic.

## Overview

This repository serves as a hub for collecting, organizing, and sharing high-quality prompts used with AI systems (LLMs, coding assistants, image generators, etc.).

## Repository Structure

```
prompts-hub/
├── README.md
├── prompts/
│   ├── coding/
│   ├── writing/
│   ├── productivity/
│   ├── data-analysis/
│   └── ...
├── templates/
└── CONTRIBUTING.md
```

- `prompts/` — Organized prompt library by category
- `templates/` — Reusable prompt templates and starters

## Usage

Browse the `prompts/` directory to find prompts by topic. Each prompt file should include:

- **Title:** Short descriptive name
- **Purpose:** What the prompt achieves
- **Prompt:** The actual prompt text
- **Tips:** Any tips for getting better results

## Contributing

Contributions are welcome. Please follow these guidelines:

1. Place prompts in the appropriate category folder
2. Use clear, descriptive filenames (e.g., `refactor-sql-query.md`)
3. Include a short title and description at the top of each file
4. Avoid duplicate or near-duplicate prompts
5. Test prompts before submitting
